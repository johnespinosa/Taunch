How to run:
1. Download flask
2. Download my project
3. Modify the UPLOAD_FOLDER variable in Taunch_Server.py so the images are uploaded correctly in this project.
4. run the runTaunch.py file




How to run tests: If you followed the previous steps, just run taunch_test.py
and taunch_test2.py
